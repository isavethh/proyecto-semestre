Estrategia Anti-MLM

NO Mostrar

  • Árboles jerárquicos de red
  • Niveles visuales (oro, diamante, platino)
  • Rankings de top afiliados públicos
  • Competencias entre embajadores
  • Datos personales de la red

SÍ Mostrar

  • Cantidad agregada por nivel
  • Impacto económico en Bs
  • Progreso personal vs. mes anterior
  • Tendencias propias
  • Métricas anónimas

Flujo de Autenticación JWT

// Estructura del Access Token (15 min) { "sub": "user_id", "tenant_id": "uuid", "branch_id": "uuid", "roles": ["TENANT_ADMIN", "AMBASSADOR"], "scopes": ["read:catalog", "write:tickets"] } // Refresh Token: 7 días, almacenado en BD (revocable) // Cambio de contexto (switch tenant) regenera tokens

Capas de Seguridad

1

Autenticación

  • JWT Access Token (15 min)
  • Refresh Token (BD)
  • MFA opcional
  • Rate limiting
2

Autorización

  • RBAC
  • ABAC para reglas complejas
  • Scopes granulares
  • Validación tenant_id
3

Aislamiento de Datos (RLS)

  • Row-Level Security PostgreSQL
  • SET LOCAL app.tenant_id
  • Políticas por tenant
4

Auditoría

  • Log de acciones sensibles
  • Quién, qué, cuándo, dónde
  • Ledger inmutable
5

Antifraude

  • Detección auto-invitaciones
  • Patrones anómalos
  • Validación dispositivos
  • Alertas automáticas

Inteligencia

Motor de IA

IA que genera valor real: comenzamos con reglas inteligentes, evolucionamos con datos

PRIORIDAD ALTA (MVP - V1)

Recomendaciones contextuales (hora, clima, ubicación)
Rankings locales dinámicos ("Top Equipetrol ahora")
Optimización de gasto (alternativas económicas)
Predicción de demanda (picos por día/hora)
Análisis de platos rentables (margen vs volumen)
Zonas calientes para conductores (mapas de calor)
Oportunidades detectadas para embajadores

PRIORIDAD MEDIA (V2 / Premium)

Sugerencias de promociones automáticas
Combinación inteligente viaje/pedido
Intent Search avanzado
Diagnóstico preliminar IA para tickets

NO IMPLEMENTAR EN V1

Chatbots conversacionales (no genera conversión)
IA emocional / avatares (complejidad sin valor)
Análisis financiero tipo ERP (overkill)
Gamificación compleja (refuerza estigma MLM)
